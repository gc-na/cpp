# C++ 네임스페이스(namespace) 가이드

## 개요
C++에서 네임스페이스(namespace)는 이름 충돌을 방지하고 코드를 조직화하는데 사용되는 방법입니다. 네임스페이스를 사용하면 변수, 함수, 클래스 등의 이름을 그룹화하여 서로 다른 컨텍스트에서 동일한 이름을 사용해도 충돌이 발생하지 않도록 할 수 있습니다.

## 문서화
C++에서 네임스페이스는 특정 이름을 범위(scope) 안에 정의하여, 같은 이름을 가진 다른 요소들과의 충돌을 피하는 기능을 제공합니다. 네임스페이스는 다음과 같은 목적을 가지고 있습니다:

1. **이름 충돌 방지**: 서로 다른 라이브러리나 모듈에서 동일한 이름의 함수나 변수가 존재할 수 있는데, 네임스페이스를 사용하면 이를 구분할 수 있습니다.
2. **코드 조직화**: 관련된 기능들을 그룹화하여 코드의 가독성과 유지보수성을 향상시킵니다.

### 사용법
네임스페이스는 `namespace` 키워드를 사용하여 정의합니다. 기본 문법은 다음과 같습니다:

```cpp
namespace NamespaceName {
    // 변수, 함수, 클래스 정의
}
```

네임스페이스를 사용하려면, 이름을 명시적으로 지정하거나 `using` 키워드를 사용할 수 있습니다:

```cpp
using namespace NamespaceName;
```

## 예제
```cpp
#include <iostream>

// 네임스페이스 정의
namespace MyNamespace {
    void display() {
        std::cout << "Hello from MyNamespace!" << std::endl;
    }
}

int main() {
    // 네임스페이스 내 함수 호출
    MyNamespace::display();
    return 0;
}
```

위의 예제는 `MyNamespace`라는 네임스페이스를 정의하고 그 안에 `display` 함수를 구현하여 호출하는 방법을 보여줍니다.

## 설명
네임스페이스 사용 시 주의해야 할 사항은 다음과 같습니다:

- **중첩 네임스페이스**: 네임스페이스는 중첩될 수 있습니다. 예를 들어, `namespace A { namespace B { ... } }`와 같이 정의할 수 있습니다.
- **using 지시어의 남용**: `using namespace`를 남용할 경우, 이름 충돌이 발생할 수 있습니다. 특정 네임스페이스의 특정 요소만 사용할 것을 권장합니다.
- **전역 네임스페이스**: 아무 네임스페이스에도 포함되지 않은 이름은 전역 네임스페이스에 속합니다. 이 경우, 다른 네임스페이스와의 충돌을 주의해야 합니다.
